Sensor Layer는 UAV에 탑재된 센서들로부터 원시 데이터를 수집하는 계층입니다. `01. 지상 정보 센터 AI`, `서비스 소개`에서는 8개 카테고리와 항목명만 간단히 정리했고, 이 문서는 각 항목별 오픈소스 데이터셋과 링크까지 포함한 상세본입니다. GPS 스푸핑 관련 항목은 항법정보에서 전자전정보로 통합했고, 음향 관련 항목은 드론상태에서 분리해 별도 카테고리로 뒀습니다(서비스소개 반영 시 확정된 내용).

**한국이어야 하는 것**: 지형/환경/도심 특성이 한국이어야 의미 있는 항목
**아니어도 되는 것**: 물리 법칙/수치 패턴이 전 세계 공통인 항목


### 무료 웹 다운

```
vworld.kr 또는 map.ngii.go.kr → DEM 다운(한국 지형)
data.go.kr/data/15003715/fileData.do → 국립공원 DEM
zenodo.org/records/15912415 → UAV Telemetry Waypoint
doi.org/10.5281/zenodo.16992975 → IDF-DS19(배터리/풍속)
kaggle.com/datasets/masud1901/military-audio-dataset-spectogram → MAD 군사 음향(포화음/총기)
kaggle.com → UAV Cyber Network Intrusion
```

### AI Hub 신청(가입 완료, 승인 대기)

```
dataSetSn=190 → 자율주행드론 비행영상(한국 4K)
dataSetSn=647 → 드론 자율항법 SLAM(다운 완료)
UAV 비행신호 데이터 → RF 신호(검색)
```
