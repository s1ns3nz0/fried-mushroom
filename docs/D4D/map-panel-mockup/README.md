# D4D 지도 / 경로 패널 목업

대시보드 `지도 / 경로` 패널의 프론트엔드 목업입니다. 위성 배경 위에 UAV 비행 경로·표적 마커·전술 격자를 렌더링합니다.

## 실행
`map-panel.html` 을 브라우저로 그냥 엽니다. 외부 의존성 없음(위성 이미지·CSS·JS 전부 단일 파일에 내장).

## 포함 기능
- **깨끗한 위성 배경** (라벨 없는 캡처를 base64로 내장) + 야간 톤(brightness/saturate 그레이딩 + 쿨 워시 + 비네트)
- **정북 1km 정격자** (통계격자형, 흐린 흰색) — 헤더 격자 버튼으로 on/off
- **10자리 MGRS 좌표** — 마커별 좌표 + 마우스 커서 실시간 좌표(좌상단)
- **마커** — 출발지(LP)·경로점(WP)·UAV·목표(OBJ)·T3(위협) + 주황 비행 경로 + 위협 반경
- **마커 탭 → 표적 정보 팝업** (유형·상태·좌표·거리·방위·배터리 등), ✕·ESC·빈 곳 탭으로 닫힘
- **줌**(1×~2.4×, 중심 기준) · **UAV 펄스 정지/재생**

## 코드 구조 (`map-panel.html`)
| 영역 | 내용 |
|---|---|
| `<style>` | 패널·헤더·격자·팝업 CSS |
| `var MARKERS` | 마커 데이터(정규좌표 `nx,ny` + 정보 필드) |
| `mgrs10(nx,ny)` | 정규좌표 → 10자리 MGRS (`EMIN/EMAX/NMIN/NMAX` 프레임, 7km×3.8km) |
| `drawGrid()` | 정북 1km 격자 (`CELL_M`) |
| `draw()` | 위성·격자·경로·마커 렌더 |
| `showPopup()` | 마커 탭 표적 정보 팝업 |

## 실서비스 이식 메모
- 배경은 현재 **정적 위성 이미지**(핀치 줌 불가). 실서비스는 **VWorld/Esri 위성 타일 레이어**로 교체 권장.
- 마커/경로의 정규좌표 투영 `S(nx,ny)` 를 지도 라이브러리의 `map.project(lng,lat)` 로 치환하면 팝업·히트테스트 로직 그대로 재사용 가능.
- MGRS는 목업용 선형 근사. 실서비스는 위경도 → UTM-K(EPSG:5179) → MGRS 변환 라이브러리(`mgrs` npm 등) 사용.
- 위치 기준: 강원 고성 화진포 일대 (캡처 프레임 중심 약 N38.268 E128.438).
